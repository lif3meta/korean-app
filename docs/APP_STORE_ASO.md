# App Store Metadata — ASO Optimized

> Apple Search Optimization (ASO): App Name, Subtitle, and Keywords are indexed for search.
> Description is NOT indexed for search but affects conversion rate.
> Every character counts.

---

## App Name (30 chars max)
```
Lzy Learn Korean
```
**Why:** "Learn Korean" is the #1 search term. Brand name + primary keyword. Short and memorable.

---

## Subtitle (30 chars max)
```
Master Korean the Fun & Lazy Way
```
**Why:** Captures user intent with "Master Korean" plus the brand personality "Fun & Lazy Way". Differentiates from competitors.

**Alternatives to A/B test later:**
- `Korean Language & Hangul Study` (28 chars)
- `Speak, Read & Write Hangul` (26 chars)

## Keywords (100 chars max, comma-separated, NO spaces after commas)
```
korean,hangul,language,learn,kpop,kdrama,vocabulary,grammar,flashcards,speak,pronunciation,alphabet,study,beginner,phrases
```
**(Exactly 100 characters)**

**Keyword strategy:**
- Do NOT repeat words already in App Name or Subtitle (Apple indexes those automatically)
- Single words only (Apple combines them — "korean" + "study" = "korean study")
- Mix high-volume + low-competition terms

| Keyword | Why |
|---|---|
| korean | Core term (already in title but reinforces) |
| hangul | Specific alphabet searchers |
| language | Broad "language learning" matches |
| learn | High intent |
| kpop | Massive search volume, K-pop fans learn Korean |
| kdrama | Same — K-drama watchers want to learn |
| vocabulary | Feature-specific |
| grammar | Feature-specific |
| flashcards | Study method searchers |
| speak | Speaking practice searchers |
| pronunciation | Specific skill |
| alphabet | "Korean alphabet" queries |
| study | Broad education intent |
| beginner | Captures "Korean for beginners" |
| phrases | "Korean phrases" is high-volume |

## ASO Tips for After Launch

1. **First 3 days matter** — Ask friends/family to download + rate
2. **Respond to all reviews** — Boosts ranking
3. **Update every 4-6 weeks** — Active apps rank higher
4. **A/B test subtitle** — Try different keyword combos via App Store Connect
5. **Localize** — Korean + Japanese + Chinese markets are huge for this app
6. **Track keywords** — Use AppFollow or Sensor Tower to monitor rankings
7. **Promotional text** — Update frequently (no build needed) for seasonal hooks
